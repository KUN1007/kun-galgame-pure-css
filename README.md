# kungal-pure-css



## 介绍



KUNGalgame 世界上最萌的 Galgame 论坛！为营造最良好的 Galgame 讨论环境为目的进行努力！



这是 KUNGalgame 网站页面的纯 html + css 实现，除了登录注册页面（sign-in.html）和编辑页面（edit.html）之外，没有用到一行 js！这里用到 js 的目的仅仅是创建基本的编辑器体。
